# 프로그래머스 LV-2 최댓값과 최솟값

공백과 같이 특정 문자를 기준으로 문자열을 자르는 방법엔 split 함수를 주로 사용한다.
여기서 최댓값과 최솟값을 구하는 방법에 정렬한 후 처음 끝을 뽑아도 되고 하나씩 비교하여 찾아도 되지만,
내장함수를 이용해서 해결하는 Math.min 과 Math.max 방법을 선택했다.

하나 중요한 점은 Math.min()의 정의가 아래와 같이 표현된다. 그렇기때문에 배열인 nums를 spread 연산자를 통해 나타내주어야 한다!

#### Syntax
```
  Math.min([value1[, value2[, ...]]])
  value1, value2, ...
  숫자형
```

====================================

최종코드 

```js
function solution(s) {
    const nums = s.split(" ");
    return `${Math.min(...nums)} ${Math.max(...nums)}`
}
```
